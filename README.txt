PhotoSudoku — README
====================
AIT102 Group Project | Academic Session 202604

---

WHAT THIS PROGRAM DOES
-----------------------
PhotoSudoku lets you photograph a printed or digital Sudoku puzzle,
automatically reads the grid, and lets you play interactively.
You can request hints for individual cells or reveal the full solution.


REQUIREMENTS
------------
- Python 3.10 or higher
- The dependencies listed in requirements.txt

Install all dependencies with:
    pip install -r requirements.txt


HOW TO RUN
----------
From the project root folder (YourgroupID_FinalProject/), run:

    streamlit run main.py

Your browser will open automatically at http://localhost:8501


FIRST-TIME SETUP (training the model)
--------------------------------------
The trained model file (model/digit_model.h5) must exist before the app
can recognise digits. It is NOT included in the ZIP due to file size limits.

To train it yourself:

  Step 1 — Place the dataset folders inside the project root:
    data/
      train/
        images/     (puzzle image files)
        metadata.jsonl
      val/
        images/
        metadata.jsonl
      test/
        images/
        metadata.jsonl

  Step 2 — Extract cell images from the dataset:
    python model/prepare_cells.py

    This creates data/cells/ with 81-cell crops for every puzzle image,
    sorted into class folders (0/ through 9/).

  Step 3 — Train the CNN:
    python model/train_model.py

    This saves model/digit_model.h5 and model/training_plots.png.
    Training takes roughly 5-15 minutes depending on your hardware.

  Step 4 — Launch the app:
    streamlit run main.py

KNOWN ISSUES / LIMITATIONS
---------------------------
- Grid detection works best with clear, straight-on photos and good lighting.
  Heavily rotated or blurry photos may fail to detect the grid.
- The app prompts you to retake the photo if detection fails.
- The model file is large (~5 MB) and is not included in the submission ZIP.
  Download/train instructions are above.
